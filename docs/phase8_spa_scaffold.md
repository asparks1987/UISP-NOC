# Phase 8 Item 1 - SPA Scaffold

## Stack
- React + Vite, React Query, WebSocket/SSE client, OIDC PKCE.

## Features
- Routing, auth guard, global error banner with request_id and next steps.
- Service worker for push/offline cache.

## Error/Diagnostics
- Banner shows connectivity (API/WS/push), last error message, and copy-diagnostics button.
